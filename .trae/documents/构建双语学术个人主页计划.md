# 构建双语学术个人主页计划

我们将基于 Vue 3 + Vite 构建一个现代、简洁的单页应用（Single Page Application）。

## 1. 依赖安装与配置
我们需要引入以下库来实现核心功能：
- **vue-i18n**: 用于实现中英文无缝切换。
- **lucide-vue-next**: 一套精美的图标库（用于社交媒体、邮箱等图标）。
- **Sass (可选)**: 用于更方便地编写 CSS 样式。

## 2. 目录结构设计
我们将采用组件化的方式开发，以保证代码清晰易维护：

```
src/
├── assets/             # 存放图片（头像、项目截图等）
├── locales/            # 语言包
│   ├── en.json         # 英文内容
│   └── zh.json         # 中文内容
├── components/         # 组件
│   ├── Sidebar.vue     # 左侧栏：头像、个人信息、联系方式
│   ├── Navbar.vue      # 顶部导航栏 + 语言切换按钮
│   ├── SectionAbout.vue   # 自我介绍
│   ├── SectionNews.vue    # 新闻动态
│   ├── SectionPubs.vue    # 学术发表
│   └── SectionGallery.vue # 设计作品展示
├── App.vue             # 主布局文件
└── main.js             # 入口文件（配置 i18n）
```

## 3. 详细实施步骤

### 第一步：基础环境搭建
1. 安装 `vue-i18n` 和 `lucide-vue-next`。
2. 配置 `src/main.js` 和 `src/i18n.js` 以启用多语言支持。

### 第二步：核心布局开发 (App.vue)
1. 创建响应式布局：
   - **桌面端**：左侧固定/跟随滚动 Sidebar，右侧为主要内容区。
   - **移动端**：顶部 Sidebar（简化版），下方为内容。
2. 实现平滑滚动（Smooth Scroll）：点击导航栏标签自动定位到对应板块。

### 第三步：组件开发
1. **Sidebar**: 包含头像（圆形/圆角）、姓名、职位、学校/机构、邮箱、社交链接（GitHub, LinkedIn 等）。
2. **Navbar**: 悬浮在内容区顶部，包含：
   - 导航项：Homepage, AboutMe, News, Publications, CV, Design Gallery。
   - 语言切换开关（中/En）。
3. **内容板块**:
   - **About**: 简单的富文本介绍。
   - **News**: 时间轴风格或列表风格的动态更新。
   - **Publications**: 经典的学术论文引用格式，包含 PDF/Code 链接。
   - **Design Gallery**: 网格布局展示设计作品缩略图。

### 第四步：多语言内容填充
1. 在 `en.json` 和 `zh.json` 中定义所有文本字段。
2. 确保切换语言时，页面内容即时更新。

## 4. 用户需准备的素材
在开发完成后，你需要替换以下内容：
- `src/assets/avatar.jpg`: 你的个人头像。
- `src/locales/*.json`: 具体的个人简介、新闻和论文列表文本。
